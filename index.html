TIM-Rec: Explicit Sparse Feedback on Multi-Item Upselling Recommendations in an Industrial Dataset of Telco Calls

A telco dataset for upselling recommendations

About the Dataset

Our dataset was collected from real-world telecommunications interactions. This dataset has been designed to capture the complexity of recommendation scenarios where multiple items are presented simultaneously and user feedback is sparse.

Dataset Schema

The dataset is provided as CSV files with the following columns:

  • user_id: Anonymized user identifier
  • contact_date: Interaction day
  • category: Item Category
  • contact_outcome: Explicit user feedback (Accepted, Refused)
  • PCAFeat_0: PCA Features (0 to 63)
  • item_id: Anonymized item offered
    user_id,contact_date,category,contact_outcome,PCAFeat_0,...,PCAFeat63,item_id
    12345,2025-01-23,Entertainment,Refused,-0.361584,...0.056747,1
